Flexbox

What is Flexbox?

What is Flexbox?

  • The Flexbox Layout (Flexible Box) module (CR stage on W3C)
  • Set of rules for container and items positioning
  • One direction
  • Made small-scale layouts or seperate components layout

Why do we need another layout tool?

  • Better grids
  • Vertical centering
  • Less css code
  • No hacks (negative margins, transforms, etc...)

What is flexbox

  • main axis -primary axis for layout
  • main-start | main-end
  • main size - width or height of parent
  • cross axis
  • cross-start | cross-end - items fill container starting with cross-start towards cross-end
  • cross size - height or width of parent

Child properties

Order

  • By default, flex items are laid out in the source order.
  • However, the order property controls the order in which they appear in the flex container.

Child properties

flex-grow

  • This defines the ability for a flex item to grow if necessary.

Child properties

flex-shrink

  • This defines the ability for a flex item to shrink if necessary.

Child properties

flex-basis

  • This defines the default size of an element before the remaining space is distributed.

Child properties

flex

  • Shorthand property for flex-grow, flex-shrink and flex-basis
  • Sets the values intelligently

Child properties

align-self

  • flex-start
  • flex-end
  • center
  • stretch
  • baseline
